# Feature Map

Accepted feature-level interactions (micro-journeys) discovered by subagents.

## Interaction Contract

For each interactive unit (`role + route + state + selector + action`):

1. Control is visible and enabled.
2. Interaction is executed.
3. Expected effect is asserted.
4. Newly revealed UI is recrawled and added to frontier.
