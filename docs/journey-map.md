# Journey Map

Accepted end-to-end journeys discovered by subagents.

## Completion Gates

- Route coverage target: `95%` of expected routes (if `expectedRoutes` configured)
- Role coverage target: all configured roles must have at least one accepted journey
- Frontier must be empty
- No-new-findings streak must meet threshold (default: `3` rounds)
